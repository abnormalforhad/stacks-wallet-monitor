# Stacks Wallet Monitor

A Python-based automation tool designed to track STX balances and transactions on the Stacks blockchain.

## Features
- Connects to the Hiro API (Stacks Mainnet).
- Monitors real-time wallet balances.
- Useful for tracking specific Stacks addresses or building trading bots.

## Tech Stack
- **Language:** Python
- **Ecosystem:** Stacks (Bitcoin Layer 2)
- **Category:** DeFi / Automation
